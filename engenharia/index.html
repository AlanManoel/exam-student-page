Engenharia de Software

Sobre a matéria de Engenharia de Software ela tem como objetivo desenvolver percepçao de elaborar e manter especificações de requisitos de software em conformidade com necessidades de diferentes tipos de projetos e restrições e desenvolver percepção clara de qualidade aplicada a produto, projeto ou processo de software, bem como seus artefatos intermediários (entregáveis ou não).

Ementa

  • Engenharia de requisitos
  • Técnicas de Elicitação
  • Documentação de requisito
  • Requisitos no CMMI
  • Gerência de requisitos
  • Métodos de avaliação de interfaces
  • Qualidade de Processo
  • Produtode Software
  • Verificação e Validação de Software
  • Revisão e Inspeção de Software
  • Medições e Métricas de Software
  • Estimativas e Custo de Software
